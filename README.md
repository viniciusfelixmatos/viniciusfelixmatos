## 🚀 Sobre Mim

Sou estudante de **Análise e Desenvolvimento de Sistemas** pela **Estácio** e desenvolvedor **Front-End** com foco na criação de aplicações web modernas, responsivas e centradas na experiência do usuário.

Possuo conhecimentos em **React**, **Angular**, **Vue.js**, **JavaScript**, **TypeScript** e **Node.js**, atuando principalmente no desenvolvimento de interfaces, integração com APIs REST e construção de aplicações escaláveis utilizando boas práticas de desenvolvimento.

Durante minha trajetória profissional, atuei como **Jovem Aprendiz Front-End no jornal O POVO**, onde participei do desenvolvimento e manutenção de aplicações web, adquirindo experiência em ambientes corporativos, trabalho em equipe e desenvolvimento de soluções voltadas para usuários reais.

Estou constantemente buscando aprimorar minhas habilidades técnicas e expandir meus conhecimentos em desenvolvimento web moderno, arquitetura de software e ecossistema JavaScript.
